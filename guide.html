GitHubに保存する / 保存到GitHub
注意：「适用 / 適用」ボタンを押しただけでは、ブラウザを閉じると変更が消えます。
必ず GitHubに保存 してください。
注意：点击「确认」后变更只保存在浏览器中。必须点击「保存到GitHub」才能永久保存。
1
Personal Access Token を取得する 获取 Personal Access Token
GitHubにログインして以下のURLを開きます（初回のみ）
登录GitHub，打开以下网址（仅首次需要）
https://github.com/settings/tokens/new
設定内容：
  Note（名前）: axiscore-xseries
  Expiration: No expiration（期限なし）
  Scopes: ☑ repo にチェック
  → 「Generate token」をクリック
  → 表示されたトークン（ghp_xxxx...）をコピー・保存
2
トークンを管理パネルに入力 在管理面板中输入Token
⚙ → 管理パネルの上部にトークン入力欄があります
⚙ → 管理面板顶部有Token输入框

「GitHubに保存」をクリック 点击「保存到GitHub」
保存が完了すると画面右下に「✓ 已保存到 GitHub」と表示されます
保存成功后，屏幕右下角会显示「✓ 已保存到 GitHub」
